## Installing Spark

* Install via sdk or download binary
  - Suggest sdk. 
  - Binary install: download, expand, and add bin to PATH, see https://datawookie.netlify.com/blog/2017/07/installing-spark-on-ubuntu/ 
* Requires Java8
* IDEs: 
  - Scala IDE (based on Eclipse)
  - Intellij
